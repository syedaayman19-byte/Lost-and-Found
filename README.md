# Lost-and-Found

A simple C++ program to manage lost and found items.  
Users can report lost items, found items, and view all records in a local file.

---

## Features

- **Report Lost Item**: Enter item name, location, and date.
- **Report Found Item**: Enter item name, location, and contact info.
- **View All Records**: Display all lost and found items.
- **Easy to Use**: Menu-based program that runs in the terminal.
- **File Storage**: All records are saved in `records.txt`.

---
